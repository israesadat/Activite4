Le web et internet c'est pareil :

Internet est un support de communication général mais le web est une partie des contenues circulant sur l'internet

Principe de world wide web :

Accéder à des documents textuels , situés et relies sur des machines accessibles par internet

URL :

Uniform Ressource Locator, décrivant l’emplacement d'une ressource qui contient l'adresse du serveur et de la page du serveur

Aspects techniques du web :

Les 3 mécanismes de base du web sont URL – HTML – HTTP

Page web et site web :

Une page web , c'est un document multimédia :image , texte , possibilité d’interaction, liens …

Création d’une page HTML :

Hyper Text Markup Language , un document HTML est un fichier texte , construit suivant une syntaxe et structuré par des balies - Les principes de langage HTML : Les balies : il doit comporter au moin 4 :
<html>...</html> : délimite du début et fin document
<head>...</head> : en-tête du document
<title>...</title>: titre du document
<body>...</body> : corps du document
Les paragraphes :
<h1>...</hn> : titre de niveau n :1à6
<p>...</p> : paragraphe
